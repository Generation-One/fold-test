This is the primary documentation file for Fold, a holographic memory system designed for development teams that stores and retrieves semantic knowledge using vector embeddings. It serves as the main reference guide covering system architecture, installation procedures, configuration requirements, API usage, and troubleshooting. The documentation outlines Fold's key components including an Axum-based REST API, SQLite metadata storage, Qdrant vector database integration, and multi-provider LLM services, establishing the foundational understanding needed for users and contributors to work with the system.
