This markdown document serves as a complete testing guide for Fold v2's AST-based file chunking system, which breaks source files into semantic units (functions, classes, methods) for fine-grained indexing and search. It documents 70+ contract-based tests and end-to-end HTTP integration tests covering chunk structure validation, language-specific AST parsing, vectorization, storage, and chunk-level search functionality across multiple programming languages. The guide provides detailed test categories, expected behaviors, and validation criteria for ensuring the chunking system works correctly across Rust, TypeScript, Python, Markdown and other file types.
