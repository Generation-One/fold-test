This file implements a Database class that provides a unified abstraction layer for data persistence operations, supporting connection pooling, parameterized queries, and transaction management. It serves as a core infrastructure component used by services like UserService and AuthService to perform CRUD operations and complex database transactions. The class uses a generic type-safe approach with async/await patterns and implements proper resource management through connection pooling. It follows the Repository pattern, encapsulating all database interactions behind a clean, reusable interface.
