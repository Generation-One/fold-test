This markdown document details the migration of Fold v2's security model from token-level project scoping to user/group-based project access control. It documents the architectural shift where API tokens now identify users rather than define project scope, with project access controlled through user roles and group membership. The file provides before/after comparisons of affected scripts, authentication flow diagrams, implementation details, and testing impact analysis to guide developers through the security model transition.
