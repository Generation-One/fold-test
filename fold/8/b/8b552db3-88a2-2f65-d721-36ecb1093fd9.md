This markdown document describes a major security architecture migration in the Fold v2 system, transitioning from token-level project scoping to user/group-based project access control. It details the changes made to three PowerShell scripts (token creation, comprehensive smoke tests, and chunk database tests), explains the new authentication flow where bearer tokens identify users rather than project scopes, and documents how project membership and role-based permissions now control access. The document serves as both a change summary and migration guide for developers updating code to work with the new security model.
