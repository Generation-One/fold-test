# Sample Project

A sample project for testing Fold's chunked indexing.

## Overview

This project demonstrates how Fold indexes a realistic project structure
with multiple file types including Rust source code and markdown documentation.

## Structure

```
mixed-project/
├── src/
│   ├── main.rs    # Application entry point
│   └── lib.rs     # Library functions
├── docs/
│   └── guide.md   # User guide
└── README.md      # This file
```

## Usage

Build and run:

```bash
cargo build
cargo run
```
